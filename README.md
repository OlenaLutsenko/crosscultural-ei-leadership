# Cross-Cultural Emotional Intelligence and Leadership

## Overview

This project investigates the relationship between Emotional Intelligence (EI) and Global Transformational Leadership (GTL) among women leaders from different cultural regions.

## Methods

- Data cleaning
- Descriptive statistics
- Spearman correlations
- Kruskal–Wallis tests
- Dunn–Holm post-hoc tests
- OLS regression
- Seaborn visualizations

## Technologies

- Python
- Pandas
- SciPy
- Statsmodels
- Seaborn
- Matplotlib

## Main Findings

- EI significantly predicts transformational leadership (R² = 0.47)
- Self-awareness is the strongest predictor of GTL
- Significant differences in EI were observed between North America and Western Europe
